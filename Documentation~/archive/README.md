# Governed Archive

Status: Active historical-support folder  
Role: Retain historical documents that still matter for traceability but are not active authority.

## What belongs here
- closed roadmaps that still matter for traceability
- historical technical-support documents repeatedly referenced during migration
- split-source or superseded design bibles worth keeping close at hand
- explicit curation ledgers that explain what happened to older documentation

- governed reference surfaces (`reference/`)
